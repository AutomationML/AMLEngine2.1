IExternalReferenceResolverLoadOrGetReferencedDocument(ExternalReferenceType) Method

Loads a CAEXDocument, referenced by the external reference. The new CAEXDocument is a separate document and no libraries are merged to the current document. This Method doesn't change any references, which use the alias of this external reference. Use the IMerge service to achieve this.
