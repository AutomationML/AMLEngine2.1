LibraryVersion Properties

The PublicLibraryCatalogLibraryVersion type exposes the following members.

Properties
 NameDescription
CAEXVersion Gets and sets the CAEX version of the library allowing different signatures for libraries, supporting differen CAEX versions but have identical content
Description Gets and sets a description text to explain the library version
DownloadLink Gets and sets the download link where to load the library. The link should be public availabe
NetVersion Gets the version converted to the .NET Version type.
Revision A revision of this version exists, using the sam version number
Signature Gets the signature (checksum) used to validate the library content
Sources Gets and sets the sources, where this version is published
Version Gets and sets the version string of the library
