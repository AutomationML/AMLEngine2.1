AmlObjectsExtensions Methods

The AmlObjectsExtensions type exposes the following members.

Methods
 NameDescription
AMLAttributes Conversion of any object which has an attribute collection to get access to specific AutomationML defined attributes.
AMLFacet Converts an InternalElementType to an AMLFacet.
AMLFile Converts a CAEX file to an AML file.
AMLGroup Converts an InternalElementType to an AMLGroup.
AMLPort Converts an InternalElementType to an AMLPort.
AMLSystemUnitClass Converts an SystemUnitClassType to an AMLSystemUnitClass.
AutomationMLVersion(CAEXDocument) Gets the AutomationML version for this CAEXFile object.
AutomationMLVersion(CAEXFileType) Gets the AutomationML version for this CAEXFile object.
ClearMetaInformation Removes all "Writer Header" specific meta information out of this CAEX file.
DeleteMetaInformation This function deletes any existing meta information corresponding to the writerID.
FrameAttribute Gets a defined FrameAttribute from the Attribute collection of the IObjectWithAttributes. If no such attribute exists, null is returned.
GetMetaInformation(CAEXFileType) This methods returns an Enumeration of all "WriterHeader" meta information out of an existing AutomationML file.
GetMetaInformation(CAEXFileType, String) This function returns the MetaInformation corresponding to the writerID, otherwise null.
IsAMLFacet Determines whether the InternalElement instance is an AMLFacet.
IsAMLGroup Determines whether the InternalElement instance is an AMLGroup.
IsAMLObject Determines whether the CAEX object is an AutomationML object. If the CAEX object is a SystemUnitClass or InternalElement, it is checked, if there exists a role reference to the AutomationMLBaseRole. If the CAEX object is an ExternalInterface, it is checked, if there exists an InterfaceClass reference to an AutomationMLBase Interface.
IsAMLPort Determines whether the InternalElement instance is an AMLPort.
IsFacetAttribute Determines, if the attribute is an attribute of an AMLFacet.
IsFacetInterface Determines, if the ExternalInterface object is an ExternalInterface of an AMLFacet.
IsRefTypeAttribute Gets a value indicating whether this instance is a RefType attribute.
IsRefURIAttribute Gets a value indicating whether this instance is a RefURI attribute.
New_FrameAttribute Creates a new Frame attribute for the IObjectWithAttributes if no Frame attribute exists. The provided values are used to initialize the Frame attribute values.
RefTypeAttribute Gets the RefTypeAttribute from the objectWithAttributes if one exists.
RefURIAttribute Gets the RefURIAttribute from the objectWithAttributes if one exists.
ReplaceMetaInformation This function replaces any existing meta information corresponding to the identical oldWriterID with the new meta information defined in the newMetainformation
SetMetaInformation(CAEXFileType, MetaInformation) This method is a short version of the setMetaInformation. Meta data with identical WriterID is replaced.
SetMetaInformation(SourceDocumentInformationType, MetaInformation) Sets the properties for the specified SourceDocumentInformation object using the provided MetaInformation
SetMetaInformation(CAEXFileType, String, String, String, String, String, String, String, String, String) This function sets the meta information required from AutomationML standard. If meta information about the same writerID already exists, this meta information is replaced. These information comprise source information about the AutomationML file.
SetMetaInformationStandardAutomationMLLibrary This method labels this CAEX file as a standard AutomationML library coming from the AutomationML e.V. It removes all other meta information and should only be used from authorized persons in agreement with the AutomationML e.V.
SetSourceDocumentInformation This function sets the meta information required from AutomationML standard. If meta information about the same writerID already exists, this meta information is replaced. These information comprise source information about the AutomationML file.
ToRefTypeAttribute Converts an AttributeType to a RefTypeAttribute.
ToRefURIAttribute Converts an AttributeType to a RefUriAttribute.
