Append This method is not applicable if the owner is an AMLFacet.
(Overrides CAEXSequenceTAppend)
Append(String) Appends a new Attribute with the provided name to the sequence of Attributes, if the conditions for Facet Attributes are fulfilled. Facet Attributes are a subset of Attributes which are defined for the Parent object of an AMLFacet. An Attribute with the specified name can only be added as a Facet Attribute, if the Facets parent has an identical Attribute with that name and the Attribute has not been associated to the Facet before.
(Overrides CAEXSequenceOfCAEXObjectsTAppend(String))
