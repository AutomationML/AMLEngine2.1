CaexValueIsNumericValue Method

Determines if the type code derived from the Attribute datatype is a numeric type. The following type codes are recognized as numeric types:
  • Double
  • Float
  • Long
  • UnsignedShort
  • UnsignedLong
  • UnsignedInt
  • UnsignedByte
  • Int
  • UnsignedShort
  • Short
  • NegativeInteger
  • PositiveInteger
  • Decimal
  • NonPositiveInteger
  • NonNegativeInteger
  • Byte
  • Integer
