CAEXDocumentExtensionsFindByID Method

AutomationML 2.1 API
Tries to locate a CAEX object from the specified ID. If the ID uses an alias reference and an IExternalReferenceResolver can be located, it is tried to resolve the external reference and to get the object from the external source.
